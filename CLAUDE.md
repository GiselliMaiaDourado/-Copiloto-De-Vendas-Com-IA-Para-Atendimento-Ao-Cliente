### 2. Conteúdo para o arquivo `CLAUDE.md`
Geralmente usado em projetos que utilizam ferramentas de desenvolvimento ou guias de desenvolvimento locais (como as diretrizes de codificação do Claude/Anthropic). Como o seu projeto é conceitual, você pode usá-lo para documentar as regras de contexto:

```markdown
# 🛠️ Diretrizes de Contexto e Engenharia de Prompt

Este repositório utiliza conceitos de Engenharia de Prompt para IA conversacional.

## 📋 Regras de Contexto
* **Modelo Sugerido:** LLMs modernos (GPT-4, Claude 3.5 Sonnet ou similares).
* **Temperatura Recomendada:** `0.5` (para manter o tom jovem e autêntico, mas sem inventar dados falsos sobre a marca).
* **Base Exclusiva:** O agente deve consultar obrigatoriamente os dados contidos na pasta `knowledge/` antes de formular qualquer resposta técnica.
